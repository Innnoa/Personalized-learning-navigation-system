# 常用命令
- 初始化数据库：`./scripts/init_database.sh`
- 启动后端：`./scripts/run_backend.sh`
- 安装前端依赖：`./scripts/install_frontend_deps.sh`
- 启动前端：`./scripts/run_frontend.sh`
- 后端冒烟：`./scripts/smoke_demo_flow.sh`
- 一键演示启动/停止：`./scripts/demo_up.sh` / `./scripts/demo_down.sh`
- 一键演示验收/回收：`./scripts/demo_check.sh` / `./scripts/demo_check_down.sh`
- 后端测试：`ctest --test-dir backend/build/tests --output-on-failure`
- 常用检索：`rg`、`find`、`git grep`
- 常用构建：`cmake -S backend -B backend/build`
