# 基于路径规划算法的自主学习导航系统

本科毕业设计项目，沈阳建筑大学 · 计算机科学与技术专业。

## 技术栈

| 层 | 技术 |
|------|------|
| 前端 | Vue 3 + Vite + Cytoscape.js + Pinia |
| 后端 | Drogon (C++17) |
| 数据库 | SQLite |
| 核心算法 | C++17（图遍历、拓扑排序、多因子评分） |

## 项目结构

```
frontend/     Vue 3 前端
backend/      Drogon 后端（Controller → Service → Repository / Planner）
database/     SQLite 初始化脚本与种子数据
scripts/      启动、测试、数据库初始化脚本
```

## 系统功能

### 学生端

- 课程知识图谱浏览（多层下钻）+ 节点详情
- 学习路径规划（目标选择 → 拓扑排序 → 多因子评分排序）
- 推荐理由解释（整体说明 + 节点级解释）
- 练习检验与反馈闭环（完成状态评定 → 掌握度更新 → 路径重算）
- 反馈回退（撤销最近一次检验结果）
- 推荐资源联动（按节点推荐视频/文章，支持上层资源继承）
- 细化学习导航（课程级 → 局部范围下钻）
- 学习者画像（掌握度分布、反馈历史、学习趋势）
- 路径导出（TXT 格式）
- 多课程支持（学生可被分配多门课程，导航栏课程切换）
- 账号注册（学生/教师）

### 教师端

- 课程管理（创建课程、查看已分配课程）
- 知识点 CRUD（行内编辑 + 新建 + 删除）
- 依赖关系编辑（复选框面板，全量替换保存）
- 学习资源管理（按知识点增删，整体保存至 JSON）
- 题库管理（单选题增删改，整体保存至 JSON，支持同步至前端）
- 学生进度查看（列表筛选、掌握度分布、个体详情）
- 学员分配（批量添加学生至课程，支持多课程分配）

### 管理员端

- 用户管理（创建、启用/禁用、角色分配）
- 课程管理（编辑元信息、删除）
- 教师分配（指派/撤销课程授课权限）
- 学习者管理（全局查看、编辑、转课）
- 操作日志（JSON 文件记录关键操作，时间倒序查看）

## 本地运行

```bash
# 初始化数据库
./scripts/init_database.sh

# 启动后端 (port 8080)
./scripts/run_backend.sh

# 安装前端依赖
./scripts/install_frontend_deps.sh

# 启动前端开发服务器
./scripts/run_frontend.sh
```

一键演示环境：
```bash
./scripts/demo_up.sh      # 启动
./scripts/demo_down.sh    # 停止
```
