# 课程直链资源优先 Requirement

## 目标

把超星课程内已确认的小章直链接入现有推荐资源系统，作为第一顺位补充资源。

## 范围

- 主路径推荐资源：覆盖当前 `backend/config/learning_resources.json` 已配置的 14 个顶层知识点。
- 细化学习推荐资源：只接入“已明确一一对应”的细化节点课程直链，不做运行时模糊猜测。
- 缺失项：输出一份当前未接入课程直链的知识点清单，明确哪些节点暂无可靠对应课程。

## 约束

- 不引入运行时模糊匹配，避免误配。
- 不把 279 条课程直链直接混入现有 `learning_resources.json`。
- 课程直链需保留为独立配置文件，便于后续维护和增补。
- 课程直链在资源排序中必须稳定排到第一位。

## 验收标准

- 对已配置课程直链的知识点，`learningResources[0]` 为超星课程原文章节。
- 顶层资源推荐和细化学习资源推荐都能返回课程直链。
- 未配置课程直链的知识点不应误注入错误链接。
- 输出缺失清单，说明当前未接入的顶层 / 细化节点。
