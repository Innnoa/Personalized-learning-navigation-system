# 答辩演示脚本

适用项目：`Personalized-learning-navigation-system`

## 演示前准备

```bash
cd /home/zazaki/Projects/Personalized-learning-navigation-system
./scripts/init_database.sh
./scripts/demo_up.sh
```

## 演示流程

### 1. 登录展示（30 秒）
- 打开 `http://localhost:5173/login`
- 使用演示账号按钮快速登录三种角色

### 2. 学生主链路（3 分钟）
- 学生登录 → 进入学习图谱，展示分层知识和掌握度着色
- 双击节点下钻至细化图谱，面包屑返回
- 点击"设为学习目标"→ 跳回首页自动生成路径
- 讲解路径总说明（为什么这样安排）
- 练习检验 → 提交反馈 → 查看路径前后对比
- 撤销反馈 → 展示回退能力
- 进入学习者画像查看掌握度变化

### 3. 教师功能（1 分钟）
- 教师登录 → 展示课程列表
- 进入课程 → 知识点编辑、依赖关系管理
- 学习资源与题库管理
- 学生列表 → 进度查看 → 增添学生

### 4. 管理员功能（1 分钟）
- 管理员登录 → 用户管理（创建/禁用/角色）
- 课程管理 → 教师分配
- 学习者管理 → 操作日志

### 5. 注册与多课程（30 秒）
- 点击"注册" → 注册学生账号
- 教师将该学生分配至两个课程
- 学生登录 → 导航栏课程切换下拉框

## 演示后回收

```bash
./scripts/demo_down.sh
```
