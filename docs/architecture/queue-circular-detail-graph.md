# 循环队列第三层图谱说明

## 1. 目标

本文件用于说明“`queue-circular` / 循环队列”这一第三层节点的来源、拆分依据与依赖关系设计。

本次目标是把“队列”二级图谱中的“循环队列”继续拆到“假溢出、首尾相接、判空判满与取模更新”这一层，让循环队列最核心的实现思想能够被单独展示。

## 4. 设计价值

这一层适合答辩时说明：

- 为什么循环队列不是新结构，而是对顺序队列空间复用问题的修正
- 为什么判空和判满是循环队列最容易讲乱、也最值得展示的点
- 为什么这一层非常适合用图谱节点来拆开解释实现规则
