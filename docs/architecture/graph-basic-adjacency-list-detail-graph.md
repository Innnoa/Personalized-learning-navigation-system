# 邻接表存储第三层图谱说明

最后更新：2026-03-19

## 1. 目标

本文件用于说明“`graph-basic-adjacency-list` / 邻接表存储”这一第三层节点的来源、拆分依据与依赖关系设计。

本次目标是把“图的存储与遍历”二级图谱中的“邻接表存储”继续拆到“定义、顶点表/边表、无向图与有向图特点、建表和邻接点访问”这一层，让 BFS/DFS 背后的存储表示更容易被解释。

## 3. 第三层节点

当前拆成以下 8 个节点：

1. 邻接表定义
2. 顶点表与边表
3. 无向图存储特点
4. 有向图存储特点
5. 建表过程
6. 插入边操作

8. 应用入口

## 4. 设计价值

这一层适合答辩时说明：

- 为什么图的遍历算法离不开底层存储表示
- 为什么邻接表更适合稀疏图，也更贴近课程里后续图算法的实现
- 为什么补上这一层后，图章节的表达不再只有“遍历过程”，而是有了“表示 -> 遍历”的完整逻辑

## 5. 当前边界

本次只新增展示层第三层图谱，不改变路径规划算法和数据库结构。
