# 构造过程第三层图谱说明

最后更新：2026-03-19

## 1. 目标

本文件用于说明“`huffman-tree-construction-process` / 构造过程”这一第三层节点的来源、拆分依据与依赖关系设计。

本次目标是把“哈夫曼树与编码”二级图谱中的“构造过程”继续拆到“初始权值集合、最小结点选择、父结点构造、结点集更新与最终成树”这一层，使哈夫曼树的贪心构造逻辑能够被分步展示。

## 2. 材料来源

- `docs/Table_of_Contents_for_Data_Structures.pdf`

直接依据：

- 第五章 5.6 哈夫曼树与哈夫曼编码
- 哈夫曼树的构造
- 合并过程与最终形成结果

## 3. 第三层节点

当前拆成以下 8 个节点：

1. 初始权值集合
2. 最小权值选择
3. 新父结点构造
4. 结点集更新
5. 重复合并策略
6. 结束条件
7. 哈夫曼树形成
8. 应用入口

## 4. 设计价值

这一层适合答辩时说明：

- 为什么哈夫曼树构造本质上是一个反复“选最小并合并”的过程
- 为什么“构造思想”和“构造过程”需要拆开，分别表达原则与步骤
- 为什么这一层很适合拿来做图谱可视化演示，因为它天然是阶段推进式过程

## 5. 当前边界

本次只新增展示层第三层图谱，不改变路径规划算法和数据库结构。
