# 邻接矩阵存储第三层图谱说明

最后更新：2026-03-19

## 1. 目标

本文件用于说明“`graph-basic-adjacency-matrix` / 邻接矩阵存储”这一第三层节点的来源、拆分依据与依赖关系设计。

本次目标是把“图的存储与遍历”二级图谱中的“邻接矩阵存储”继续拆到“编号映射、矩阵单元、无权/带权图表示、对称性与查边特征”这一层，使图表示层不只停留在邻接表一边。

## 2. 材料来源

- `docs/Table_of_Contents_for_Data_Structures.pdf`

直接依据：

- 第六章 6.2 图的存储结构
- 邻接矩阵定义
- 无向图与带权图表示

## 3. 第三层节点

当前拆成以下 8 个节点：

1. 邻接矩阵定义
2. 顶点编号映射
3. 矩阵单元含义
4. 无权图表示
5. 带权图表示
6. 无向图对称性
7. 查边与遍历便利性
8. 应用入口

## 4. 设计价值

这一层适合答辩时说明：

- 为什么邻接矩阵和邻接表都要展开，而不是只讲一种图表示
- 为什么矩阵更适合快速查边，但遍历一个顶点的邻接点成本更高
- 为什么这一层能让图章节的“表示方法对算法影响”讲得更完整

## 5. 当前边界

本次只新增展示层第三层图谱，不改变路径规划算法和数据库结构。
