# 线性表链式表示第三层图谱说明

最后更新：2026-03-19

## 1. 目标

本文件用于说明“`linear-list-linked-representation` / 链式表示”这一第三层节点的来源、拆分依据与依赖关系设计。

本次目标是把“线性表”二级图谱中的“链式表示”继续拆到“结点、指针、访问方式和插删优势”这一层，让链式表示为什么会导向链表实现能够被单独讲清。

## 3. 第三层节点

当前拆成以下 8 个节点：

1. 结点组成
2. 指针链接关系
3. 非连续存储
4. 按序访问方式
5. 插入删除便捷
6. 额外指针开销
7. 头指针入口
8. 应用入口

## 4. 设计价值

这一层适合答辩时说明：

- 为什么链式表示不依赖连续内存
- 为什么链式表示的优势和顺序表示几乎形成对照
- 为什么这一层能自然过渡到后续“链表”二级图谱

本次只新增展示层第三层图谱，不改变路径规划算法和数据库结构。
