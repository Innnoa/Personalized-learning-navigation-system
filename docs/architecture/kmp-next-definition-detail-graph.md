## 1. 目标

本文件用于说明“`kmp-next-definition` / next 数组含义”这一第三层节点的来源、拆分依据与依赖关系设计。

本次目标是把“KMP 模式匹配”二级图谱中的“next 数组含义”继续拆到“前后缀、失配位置、next[j] 含义、回退位置与示例推导”这一层，把 KMP 最难解释的核心抽象讲清楚。
