# 非递归遍历思想第三层图谱说明

最后更新：2026-03-19

## 1. 目标

本文件用于说明“`binary-tree-traversal-iterative` / 非递归遍历思想”这一第三层节点的来源、拆分依据与依赖关系设计。

本次目标是把“二叉树遍历”二级图谱中的“非递归遍历思想”继续拆到“栈/队列替代递归、访问状态维护、三类非递归框架与实现对比”这一层，让递归与非递归的差异在答辩演示中更容易讲清。

## 3. 第三层节点

当前拆成以下 8 个节点：

1. 栈或队列替代递归
2. 访问状态维护
3. 先序非递归框架
4. 中序非递归框架
5. 后序非递归难点
6. 层序遍历与队列
7. 递归与非递归对比
8. 应用入口

## 4. 设计价值

这一层适合答辩时说明：

- 为什么“非递归”不是简单把递归改写成循环，而是把隐式调用栈显式化
- 为什么中序和后序的非递归实现比先序更值得展示
- 为什么这一层和“递归遍历思想”可以形成一组很清楚的对照关系
