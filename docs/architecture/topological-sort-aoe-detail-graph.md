# AOE 网第三层图谱说明

最后更新：2026-03-19

## 1. 目标

本文件用于说明“`topological-sort-aoe` / AOE 网”这一第三层节点的来源、拆分依据与依赖关系设计。

本次目标是把“拓扑排序与关键路径”二级图谱中的“AOE 网”继续拆到“建模目标、事件/活动区分、边权含义、源点汇点与工程调度语义”这一层，让关键路径问题的建模入口单独变得可讲解。

## 2. 材料来源

直接依据：

- 第六章 6.6.2 关键路径
- AOE 网建模
- 事件、活动和持续时间语义

## 3. 第三层节点

当前拆成以下 8 个节点：

1. AOE 网建模目标
2. 事件与活动区分
3. 边权表示持续时间
4. 源点与汇点
5. 关键路径分析前提
6. 与 AOV 网区别
7. 工程调度语义
8. 应用入口

## 4. 设计价值

这一层适合答辩时说明：

- 为什么关键路径问题首先是一个建模问题，而不只是一个计算问题
- 为什么 AOE 网和 AOV 网需要明确区分，否则很容易把“活动在点上”与“活动在边上”混淆
- 为什么这一层和本项目“学习路径导航”的依赖语义展示天然相通

本次只新增展示层第三层图谱，不改变路径规划算法和数据库结构。
