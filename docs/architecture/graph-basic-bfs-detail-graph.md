# 广度优先遍历第三层图谱说明

最后更新：2026-03-19

## 1. 目标

本文件用于说明“`graph-basic-bfs` / 广度优先遍历”这一第三层节点的来源、拆分依据与依赖关系设计。

本次目标是把“图的存储与遍历”二级图谱中的“广度优先遍历”继续拆到“层次策略、队列支持、访问标记、出队访问与邻接点入队”这一层，让 BFS 与 DFS 的差异能够被直观看见。

## 2. 材料来源

- `docs/Table_of_Contents_for_Data_Structures.pdf`

直接依据：

- 第六章 6.3.2 广度优先遍历
- 图广度优先遍历的实现要点
- 邻接矩阵与邻接表下的 BFS

## 3. 第三层节点

当前拆成以下 8 个节点：

1. BFS 目标与策略
2. 队列支持
3. 访问标记
4. 出队访问当前顶点
5. 入队未访问邻接点
6. 邻接矩阵下 BFS
7. 邻接表下 BFS
8. 应用入口

## 4. 设计价值

这一层适合答辩时说明：

- 为什么 BFS 的核心不是“遍历”两个字，而是“按层推进”
- 为什么 BFS 和队列之间存在稳定的一一对应关系
- 为什么补上这一层后，系统就能对比展示 DFS 与 BFS 两类图搜索过程

## 5. 当前边界

本次只新增展示层第三层图谱，不改变路径规划算法和数据库结构。
