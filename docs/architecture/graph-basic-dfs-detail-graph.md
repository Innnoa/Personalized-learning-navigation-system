# 深度优先遍历第三层图谱说明

最后更新：2026-03-19

## 1. 目标

本文件用于说明“`graph-basic-dfs` / 深度优先遍历”这一第三层节点的来源、拆分依据与依赖关系设计。

本次目标是把“图的存储与遍历”二级图谱中的“深度优先遍历”继续拆到“访问标记、递归框架、矩阵/邻接表下 DFS、连通分量与 DFS 树”这一层，让图搜索过程变得可讲解、可展示。

## 2. 材料来源

直接依据：

- 图深度优先遍历的实现要点

- 邻接矩阵与邻接表下的 DFS

## 3. 第三层节点

当前拆成以下 8 个节点：

1. DFS 目标与策略
2. 访问标记机制
3. 递归实现框架
4. 邻接矩阵下 DFS
5. 邻接表下 DFS
6. 连通分量遍历
7. DFS 树与搜索轨迹
8. 应用入口

## 4. 设计价值

这一层适合答辩时说明：

- 为什么图搜索要比树搜索更强调“访问标记”
- 为什么 DFS 和“递归工作栈”存在天然呼应
- 为什么第三层图谱可以把算法流程拆成更容易讲解的若干关键步骤

## 5. 当前边界

本次只新增展示层第三层图谱，不改变路径规划算法和数据库结构。
