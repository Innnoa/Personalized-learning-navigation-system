## 1. 目标

本文件用于说明“`huffman-tree-code-decode` / 哈夫曼编码与译码”这一第三层节点的来源、拆分依据与依赖关系设计。

本次目标是把“哈夫曼树与编码”二级图谱中的“哈夫曼编码与译码”继续拆到“编码生成、前缀无歧义、按位译码和压缩效果”这一层，使哈夫曼树从构造走向应用的过程完整闭合。

## 3. 第三层节点

当前拆成以下 8 个节点：

1. 编码路径约定
2. 叶结点对应字符
3. 根到叶生成编码
4. 前缀无歧义
5. 编码结果组织
6. 按位译码过程
7. 压缩效果理解
8. 应用入口

## 4. 设计价值

这一层适合答辩时说明：

- 为什么哈夫曼树不是只停留在“构造出来”，而是必须走到编码与译码才算完整
- 为什么前缀无歧义是译码能够顺畅进行的核心条件
- 为什么这一层很适合用来展示“结构 -> 编码 -> 应用”的完整知识链
