# 头插法建表第三层图谱说明

最后更新：2026-03-19

## 1. 目标

本文件用于说明“`linked-list-head-insert` / 头插法建表”这一第三层节点的来源、拆分依据与依赖关系设计。

本次目标是把“链表”二级图谱中的“头插法建表”继续拆到“新结点申请、指针重连、头指针更新与逆序特征”这一层，让链表建表过程能够按步骤清楚展示。

## 3. 第三层节点

当前拆成以下 8 个节点：

1. 头结点与首元结点
2. 新结点申请
3. 新结点数据赋值
4. 新结点指向原首元
5. 头结点改指向新结点
6. 重复插入形成链表
7. 逆序特征理解
8. 应用入口

## 4. 设计价值

这一层适合答辩时说明：

- 为什么链表建表的核心是指针连接顺序，而不是元素搬移
- 为什么头插法特别适合展示“先接旧链，再改头指针”的实现细节
- 为什么头插法天然带有逆序特征，适合与尾插法形成对照
