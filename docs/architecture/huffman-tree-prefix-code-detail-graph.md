# 前缀编码约束第三层图谱说明

最后更新：2026-03-19

## 1. 目标

本文件用于说明“`huffman-tree-prefix-code` / 前缀编码约束”这一第三层节点的来源、拆分依据与依赖关系设计。

本次目标是把“哈夫曼树与编码”二级图谱中的“前缀编码约束”继续拆到“前缀定义、歧义来源、唯一可译性和应用价值”这一层，让哈夫曼编码为什么可用这件事真正讲清。

## 2. 材料来源

- `docs/Table_of_Contents_for_Data_Structures.pdf`

直接依据：

- 第五章 5.6 哈夫曼树与哈夫曼编码
- 前缀编码性质

## 3. 第三层节点

当前拆成以下 8 个节点：

1. 前缀编码定义
2. 非前缀编码歧义
3. 哈夫曼编码满足前缀性
4. 叶结点对应字符
5. 唯一可译性
6. 译码无需分隔符
7. 压缩与可译平衡
8. 应用入口

## 4. 设计价值

这一层适合答辩时说明：

- 为什么哈夫曼编码不是只看“短”，还必须看“能不能稳定译码”
- 为什么前缀性和叶结点承载字符之间存在直接关系
- 为什么这一层能把哈夫曼章节从“构造过程”延伸到“编码正确性”
