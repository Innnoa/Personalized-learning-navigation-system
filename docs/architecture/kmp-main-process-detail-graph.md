# KMP 主过程第三层图谱说明

最后更新：2026-03-19

## 1. 目标

本文件用于说明“`kmp-main-process` / KMP 主过程”这一第三层节点的来源、拆分依据与依赖关系设计。

本次目标是把“KMP 模式匹配”二级图谱中的“KMP 主过程”继续拆到“双指针推进、失配按 next 回退、主串不回退与完整匹配判定”这一层，让 KMP 的核心运行逻辑能够被逐步展示。

## 2. 材料来源

直接依据：

- 第四章 4.3 串的模式匹配

## 3. 第三层节点

当前拆成以下 8 个节点：

1. 初始化双指针
2. 当前字符比较
3. 匹配后同时后移
4. j=0 特殊处理
5. 按 next 回退 j
6. 主串指针不回退
7. 完整匹配判定
8. 应用入口

## 4. 设计价值

这一层适合答辩时说明：

- 为什么 next 数组不是终点，真正难讲的是它如何驱动主过程
- 为什么 KMP 的核心价值可以浓缩成“主串不回退”
- 为什么这一层和“next 数组含义”一起，能把 KMP 讲成完整闭环
