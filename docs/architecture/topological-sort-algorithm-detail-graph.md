# 拓扑排序算法第三层图谱说明

最后更新：2026-03-19

## 1. 目标

本文件用于说明“`topological-sort-algorithm` / 拓扑排序算法”这一第三层节点的来源、拆分依据与依赖关系设计。

本次目标是把“拓扑排序与关键路径”二级图谱中的“拓扑排序算法”继续拆到“入度、零入度顶点选择、输出序列、删除出边、更新入度与合法性判断”这一层，使第六章中最贴近本项目“依赖排序”思想的算法主干能够被完整展示。

## 3. 第三层节点

当前拆成以下 8 个节点：

1. 入度概念
2. 初始化入度表
3. 选取零入度顶点
4. 输出拓扑序列
5. 删除相关出边
6. 更新后继入度
7. 环检测与合法性判断
8. 应用入口

## 4. 设计价值

这一层适合答辩时说明：

- 为什么拓扑排序本质上就是“不断消解前置约束”的过程
- 为什么它和本项目的学习路径规划概念天然一致
- 为什么把“拓扑排序算法”和“关键路径求解”分别下钻，能够避免把两个不同问题混在一起
