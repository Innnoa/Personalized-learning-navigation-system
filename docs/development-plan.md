# 开发计划

最后更新：2026-03-17

## 1. 总体原则

- 先做能跑通的主链路，再补优化与解释能力
- 先固定课程范围与数据结构，再写路径规划算法
- 先保证算法正确性，再做前端展示细节

## 2. 推荐开发顺序

### 阶段一：确定课程范围与数据库结构

- 明确演示课程与知识点范围
- 设计 SQLite 表结构
- 编写 `database/init/001_schema.sql`
- 编写第一批 `database/seed/` 演示数据

### 阶段二：搭建后端基础能力

- 初始化 Drogon 项目
- 建立 Controller、Service、Repository、Algorithm 基本分层
- 完成健康检查接口
- 完成知识图谱查询接口

### 阶段三：完成图构建与路径规划

- 从数据库读取知识点与依赖关系
- 通过 GraphBuilder 组装 `KnowledgeGraph`
- 实现拓扑排序、依赖检查、路径生成等核心算法
- 输出 `LearningPath` 与 `PathReasonTrace`

### 阶段四：完成前端图谱展示

- 初始化 Vue 项目
- 接入 Cytoscape.js
- 展示知识点图谱、选中节点信息、推荐路径高亮

### 阶段五：完成动态调整与反馈闭环

- 设计学习反馈数据结构
- 实现 `POST /api/path/adjust`
- 实现 `POST /api/feedback/submit`
- 验证推荐路径会随反馈发生合理变化

### 阶段六：补充推荐理由解释

- 根据 `PathReasonTrace` 输出结构化解释信息
- 在前端展示“为什么推荐这个知识点/路径”
- 优化答辩演示话术与展示效果

## 3. 当前优先级

当前最优先任务：

1. 确定课程范围
2. 确定数据库表结构
3. 初始化 Drogon 与 Vue 项目骨架
4. 跑通知识图谱读取与展示主链路
